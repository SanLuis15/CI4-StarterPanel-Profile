# QUICK START GUIDE - Student Profile Page

## 🚀 3 STEPS TO GET STARTED

### STEP 1: Run SQL Migration (2 minutes)
1. Open phpMyAdmin (http://localhost/phpmyadmin)
2. Select your database
3. Click "SQL" tab
4. Open file: `profile_migration.sql`
5. Copy all SQL code
6. Paste into SQL box
7. Click "Go"
8. ✅ Done! 7 columns added to users table

### STEP 2: Test Profile Page (1 minute)
1. Start your server: `php spark serve`
2. Login to your application
3. Look at left sidebar under "SYSTEM"
4. Click "My Profile"
5. ✅ You should see your profile page!

### STEP 3: Edit Your Profile (2 minutes)
1. Click "Edit Profile" button
2. Fill in your information:
   - Student ID: 2021-00123
   - Course: BSIT
   - Year Level: 3
   - Section: IT3A
   - Phone: 0912-345-6789
   - Address: Your address
3. Upload a profile picture (optional)
4. Click "Update Profile"
5. ✅ Profile updated successfully!

---

## 📸 UPLOAD PROFILE IMAGE

1. Click "Edit Profile"
2. Click "Choose File" under Profile Image
3. Select image (JPG, PNG, WEBP only)
4. See live preview immediately
5. Click "Update Profile"
6. ✅ Image saved!

**Image Requirements:**
- Max size: 2MB
- Formats: JPG, PNG, WEBP
- Stored in: `public/uploads/profiles/`

## ⚠️ COMMON ISSUES

**"Profile link not working"**
→ Make sure you're logged in

**"Image not uploading"**
→ Check file size under 2MB
→ Check file type (JPG, PNG, WEBP)

**"Folder not writable"**
→ Check `public/uploads/profiles/` exists
→ Set permissions: `chmod 777 public/uploads/profiles/`
