DATABASE SETUP INSTRUCTIONS
===========================

Someone opening this project needs to:

1. Make sure XAMPP is installed with MySQL running

2. Create the database:
   - Open phpMyAdmin: http://localhost/phpmyadmin
   - Click "New" 
   - Database name: project
   - Click "Create"

3. Run migrations to create tables:
   cd CI4-StarterPanel-master
   php spark migrate

4. (Optional) Seed basic data:
   php spark db:seed SystemSeeder

5. Register a new account at /register

6. Start the server:
   php spark serve

7. Access: http://localhost:8080

ALTERNATIVE: Import project.sql file
- In phpMyAdmin, select 'project' database
- Click Import tab
- Choose project.sql file
- Click Go

The migrations will create all required tables automatically.
No manual SQL import needed if using migrations.
