## 🔧 TROUBLESHOOTING

### Profile link not working?
- Make sure you're logged in
- Check session has 'username' set
- Clear browser cache

### Image not uploading?
- Check `public/uploads/profiles/` folder exists
- Check folder has write permissions (777)
- Check file size is under 2MB
- Check file type is JPG, PNG, or WEBP

### Form validation errors?
- Fullname and username are required
- Username must be unique
- Year level must be 1-5
- Check all field length limits
